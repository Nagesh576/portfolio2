## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/Nagesh576/portfolio2.git
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Project

```bash
npm run build
```
